Développement d'une application web de référence pour ingénieurs tuyauterie : l'utilisateur saisit le type de bride et sa classe de pression (ASME B16.5 / EN 1092), la plateforme retourne automatiquement les diamètres de boulons, clés de serrage, couples de serrage et normes applicables.
Compétences : Normes ASME/EN · Tuyauterie industrielle · Développement web · Base de données techniques
