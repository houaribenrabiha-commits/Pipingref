CLASSES PRESSION

// ASME B16.5 — Correspondances PN/Class

Class 150

≈ PN 20 · 19.6 bar à 38°C
Applications basse pression. Utilisation courante en eau, vapeur basse pression, gaz non corrosifs. Matériau standard A105 / WCB.

Class 300

≈ PN 50 · 51.1 bar à 38°C
Applications moyennes pressions. Vapeur process, gaz de procédé. Très répandu en industrie pétrochimique.

Class 400

≈ PN 68 · 68.1 bar à 38°C
Utilisé en Europe principalement pour des applications spécifiques entre Class 300 et 600.

Class 600

≈ PN 100 · 102.1 bar à 38°C
Haute pression. Pipelines, raffineries, unités de compression gaz. Nécessite boulonnage renforcé A193 B7.

Class 900

≈ PN 150 · 153.2 bar à 38°C
Très haute pression. Installations de compression, lignes HP. Inspection radiographique obligatoire.

Class 1500

≈ PN 250 · 255.3 bar à 38°C
Domaine HP critique. Têtes de puits (wellhead), lignes de gaz haute pression. Design sur code ASME VIII.

Class 2500

≈ PN 420 · 425.5 bar à 38°C
Pression extrême. Applications offshore, circuits de test HP. Poids et coût très élevés.
